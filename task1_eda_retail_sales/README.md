# Exploratory Data Analysis on Retail Sales Data

## Objective
The aim of this project is to analyze retail sales data and identify customer purchasing patterns using Exploratory Data Analysis (EDA).

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset Features
- Transaction ID
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price per Unit
- Total Amount

## Steps Performed
1. Loaded retail sales dataset
2. Explored dataset information
3. Checked missing values
4. Performed statistical analysis
5. Created histogram visualization
6. Created bar chart visualization
7. Created pie chart visualization

## Visualizations
- Age Distribution Histogram
- Customer Spending Bar Chart
- Gender Distribution Pie Chart

## Output
Successfully analyzed retail sales data and generated meaningful visual insights.
